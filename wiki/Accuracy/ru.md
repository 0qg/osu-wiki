# Точность

Точность (*аккураси*, *акка*) – это процентное выражение способности игрока нажимать на [ноты](/wiki/Hit_object) вовремя. У игроков есть всего три точности: точность на карте, которая зависит от полученных очков за нажатия; общая точность игрока, которая зависит от всех скоров игрока (однако они взвешиваются, чтобы лучшие скоры влияли на точность больше); и точность по [очкам производительности (pp)](/wiki/Performance_Points) игрока, которая зависит от точности его лучших пп-скоров.

## Игровые режимы

### ![](/wiki/shared/mode/osu.png) osu!standard

![Accuracy = (50 \* number of 50s + 100 \* number of 100s + 300 \* number of 300s) / 300(number of 0s + number of 50s + number of 100s + number of 300s)](img/accuracy_standard.png "Формула расчета точности для osu!standard")

В osu!standard точность рассчитывается как сумма всех очков, полученных игроком за каждую ноту, разделённая на максимально возможное количество очков на карте. 

Пример с одной нотой:

```
300 -> 300 / 300 = 1   = 100.00%
100 -> 100 / 300 = 1/3 =  33.33%
50  ->  50 / 300 = 1/6 =  16.67%
0   ->   0 / 300 = 0   =   0.00%
```

### ![](/wiki/shared/mode/taiko.png) osu!taiko

![Accuracy = 0.5(number of GOOD + number of GREAT) / (number of BAD + number of GOOD + number of GREAT)](img/accuracy_taiko.png "Формула расчета точности для osu!taiko")

В osu!taiko точность рассчитывается как сумма точностей нот, разделенная на общее количество нот. Точность ноты может быть либо GREAT (良) (учитывается как 100%), GOOD (可) (учитывается как 50%), или MISS/BAD (不可) (считается за 0%, что также сбрасывает комбо). Слайдеры (drum roll) и спиннеры не влияют на точность.

### ![](/wiki/shared/mode/catch.png) osu!catch

![Accuracy = (number of droplets + number of drops + number of fruits) / (number of missed droplets + number of missed drops + number of missed fruits + number of droplets + number of drops + number of fruits)](img/accuracy_catch.png "Формула расчета точности для osu!catch")

В osu!catch точность рассчитывается как количество всех собранных объектов, разделенное на количество всех объектов (бананы при этом нигде не учитываются). Все объекты кроме бананов имеют одинаковое значение.

*Примечание для API пользователей: если вам нужно рассчитать точность в osu!catch, количество капелек объявлено как `count50`, а количество пропущенных капелек `countkatu`.*

### ![](/wiki/shared/mode/mania.png) osu!mania

![Accuracy = (50 \* number of 50s + 100 \* number of 100s + 200 \* number of 200s + 300 \* number of 300s + 300 \* number of MAXs) / 300(number of 0s + number of 50s + number of 100s + number of 200s + number of 300s + number of MAXs)](img/accuracy_mania.png "Формула расчета точности для osu!mania")

В osu!mania точность рассчитывается аналогично [osu!standard](#-osu!standard).

## График производительности

![Performance graph](img/performance_graph.jpg "График производительности")

График производительности это производительность игрока (жизненная полоса игрока) в течение игры (времени). При наведении курсора на график отображается дополнительная информация.
*Примечание: Дополнительная информация может быть отображена только после того, как вы сыграли карту или посмотрели её реплей. После выхода со [страницы результатов](/wiki/results_screen) эту информацию уже не увидеть.*

### Accuracy

При наведении курсора на график всплывает окошко со значениями `Error` (Отклонение) и `Unstable Rate` (Нестабильность).

В связи с тем, как реализованы моды [DT](/wiki/DT) (Double Time) и [HT](/wiki/HT) (Half Time), значения `Error` и `Unstable Rate` будут соответственно умножены на множитель мода. Чтобы получить истинные значения при игре с DT модом, разделите их на 1.5. Аналогично, при игре с HT модом умножьте их на 1.33.

#### Error (Отклонение)

`Error` всегда показывает два значения, которые представляют собой насколько раньше и насколько позже вы в среднем нажимаете на ноты. Чем выше [Overall Difficulty](/wiki/Overall_Difficulty) карты, тем ниже должны быть ваши значения отклонения, чтобы хорошо на ней сыграть.

#### Unstable Rate (Нестабильность)

`Unstable Rate` отражает то, как стабильно вы нажимаете на ноты, и чем меньше это значение, тем лучше (лучшие игроки часто имеют значение ниже 100). Обратите внимание, что это отображает именно стабильность, а не точность – постоянные нажатия на 1000 мс раньше дадут то же значение, что и стабильные нажатия вовремя. Формула по сути является стандартным разбросом отклонений нажатий (в миллисекундах), умноженным на 10. [Образец кода](https://gist.github.com/peppy/3a11cb58c856b6af7c1916422f668899) доступен для изучения и показывает, как в osu-stable рассчитывается Unstable Rate.

### Spin

*Примечание: Spin отображается только для [osu!standard](/wiki/Game_mode/osu!).*

В дополнение к точности, также выдается дополнительная информация о спиннерах.

#### Speed (скорость)

`Speed` представляет собой средний RPM (revolutions per minute - обороты в минуту) на всех спиннерах карты. `Max`  показывает самый высокий RPM, которого достиг игрок на карте.
