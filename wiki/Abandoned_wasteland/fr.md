# Abandoned wasteland

Le **Abandoned wasteland** est un [sous-forum](https://osu.ppy.sh/community/forums) privé utilisé pour cacher les forums et les sujets du forum de la vue de tous. Ce sous-forum est uniquement accessible aux modérateurs qui ont la possibilité de modifier les forums. Les instructions aux modérateurs le concernant sont : `Mettez les choses dont nous n'avons pas vraiment besoin ici.`.

Les forums et les sujets sont dits "Wastelanded" lorsqu'ils seront déplacés vers ce forum.
