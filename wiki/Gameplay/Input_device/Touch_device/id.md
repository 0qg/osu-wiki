# Perangkat sentuh

*See also: [Play style](/wiki/Play_style)*

**Perangkat sentuh** atau *touch device* adalah sebuah perangkat layar sentuh yang mengizinkan pengguna untuk berinteraksi dengan komputer menggunakan jari-jari tangan. osu! mendukung perangkat sentuh asli. Kamu dapat mengentuh layar untuk mengarahkan dan menekan pada saat bersamaan. Perangkat sentuh paling sering muncul dalam bentuk laptop layar sentuh, tablet grafis kelas atas, atau smartphone.

Jika perangkat sentuh digunakan untuk memainkan beatmap mode permainan osu! dan hasil permainan dikirimkan, sebuah mod yang menunjukkan penggunaan perangkat sentuh akan diterapkan secara otomatis.
