# osu!api

osu! tem uma [API](https://pt.wikipedia.org/wiki/Interface_de_programa%C3%A7%C3%A3o_de_aplica%C3%A7%C3%B5es) pública que podem ser usados por serviços terceirizados para expandir a plataforma osu!. A API costumava ser mais privada, usada apenas por aqueles que a solicitavam, mas agora qualquer um pode usar a API desde que tenha uma conta osu!.

Você pode entcontrar a documentação da API na [Wiki da osu-api](https://github.com/ppy/osu-api/wiki). A versão atual da API usa chaves privadas para autenticar requests. Você pode aplicar para uma chave da API na [página da aplicação](https://osu.ppy.sh/p/api) no site do osu!. Note que a documentação na wiki pode não ser completa ou extensiva, e a API não está em desenvolvimento ativo no momento.

---

Versão 2 da osu!api está em desenvolvimento em [osu-web](https://github.com/ppy/osu-web), e pretende ser mais robusta que a original. Documentação (sem garantia de estar atualizada) pode ser encontrada na [página de referência](https://docs.ppy.sh).

Autenticação da nova API segue o [protocolo OAuth 2](https://oauth.net/2/). Clients podem ser gerenciados pela [página de configurações da conta](https://osu.ppy.sh/home/account/edit).
