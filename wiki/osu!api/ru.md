# osu!api

У osu! есть общедоступный [API](https://ru.wikipedia.org/wiki/API) — набор методов, с помощью которых можно получить разную информацию об игроках, картах и т.д.

Вся документация к API находится на [osu!api wiki](https://github.com/ppy/osu-api/wiki).
Для доступа требуется учётная запись в osu!, после чего на странице [регистрации своего приложения](https://osu.ppy.sh/p/api) можно буджет получить приватный ключ для запросов.

**Важно:** документация к osu!api может быть неполной или неточной, как минимум потому, что его развитие приостановлено, и все силы разработчиков направлены на проект [osu-web](https://github.com/ppy/osu-web) и его API.
