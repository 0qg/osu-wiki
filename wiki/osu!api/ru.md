У osu! есть публичный API, который сторонние сервисы могут использовать для расширения и поддержки osu! платформы. Раньше он был более приватным, использовался только для каждого запроса, но теперь любой может использовать API, если у него есть учетная запись osu!.

Вы можете найти документацию по API в [osu!api Вики] (https://github.com/ppy/osu-api/wiki). Текущая версия API использует закрытые ключи для аутентификации запросов. 
Вы можете подать заявку на ключ API на [странице приложения] (https://osu.ppy.sh/p/api) на веб-сайте osu!.

Обратите внимание, что документация на вики osu!api может быть не полной и не обширной, а сам API не находится в активной разработке. Новый, гораздо более надежный API, находится в разработке на [osu-web] (https://github.com/ppy/osu-web).
