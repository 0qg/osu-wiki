# osu!api

osu! memiliki [API](https://en.wikipedia.org/wiki/API) publik yang dapat digunakan oleh siapa saja untuk memperluas dan mendukung plaform osu!. Dulunya API ini bersifat privat, dan hanya boleh digunakan jika kamu mengajukan permintaan akses, tapi sekarang siapa pun dapat menggunakan API ini selama mereka memiliki akun osu!.

Kamu dapat melihat dokumentasi API osu! melewati [wiki osu-api](https://github.com/ppy/osu-api/wiki). Versi API saat ini memerlukan kunci privat untuk mengotentikasi permintaan komunikasi. Kamu dapat mengajukan pembuatan kunci API di [laman pengajuannya](https://osu.ppy.sh/p/api) melewati situs web osu!. Perlu dicatat bahwa dokumentasi [wiki osu!api]((https://github.com/ppy/osu-api/wiki)) tidak sepenuhnya lengkap, dan API itu sendiri tidak sedang dalam pengembangan yang sangat aktif.

---

Versi 2 dari osu!api saat ini sedang dalam tahap pengembangan di [osu-web](https://github.com/ppy/osu-web), dan tujuan pengembangan versi baru ini untuk memperkuat strukturnya dibanding yang asli. Dokumentasi (tidak dijamin terbaru) dapat ditemukan di [laman referensi berikut](https://docs.ppy.sh).

Otentikasi untuk API baru mengikuti [protokol OAuth 2](https://oauth.net/2/). Klien dapat dikelola melalui [laman pengaturan akun osu!](https://osu.ppy.sh/home/account/edit) tiap pengguna.
