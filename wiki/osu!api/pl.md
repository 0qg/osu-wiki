# osu!api

osu! ma publiczne [API](https://en.wikipedia.org/wiki/API) które usługi trzecie mogą rozszerzać i wspierać platformę osu!. Kiedyś było bardziej prywatne, tylko do użytku na żądanie, ale teraz każdy może korzystać z API, o ile ma konto osu!.

Możesz znaleźć dokumentację API na [Wiki osu-api](https://github.com/ppy/osu-api/wiki). Obecna wersja interfejsu API używa kluczy prywatnych do uwierzytelniania żądań. Możesz ubiegać się o klucz API na [stronie wniosku](https://osu.ppy.sh/p/api) na stronie internetowej osu!.

Zauważ, że dokumentacja wiki osu!api może nie być całkowicie kompletna lub obszerna, a sam interfejs API nie jest aktywnie rozwijany. W [osu-web](https://github.com/ppy/osu-web) działa nowy, znacznie bardziej niezawodny interfejs API.
