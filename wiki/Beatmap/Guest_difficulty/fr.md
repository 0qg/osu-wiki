# Difficulté de l'invité

*Pour les règlements concernant les difficultés des invités, voir : [Critères de classement](/wiki/Ranking_Criteria)*

Une **difficulté invité** (ou *GD* pour faire court) est une [difficulté](/wiki/Beatmap/Difficulty) d'une [beatmap](/wiki/Beatmap) qui n'est pas créée par l'[hôte de la beatmap](/wiki/Beatmap/Beatmap_host). Ils peuvent généralement être identifiés par leur nom de difficulté contenant le nom d'utilisateur du créateur invité. Un beatmap comportant plusieurs créateurs est également appelé [difficulté collaborative](/wiki/Beatmap/Beatmap_collaborations).

Les difficultés invités sont souvent demandées en messages privés par le ou les créateurs. Bien que les GD puissent également être demandées dans les [modding queues](https://osu.ppy.sh/community/forums/60)<!-- TODO: should link to an article about queues, not the forum -->.
