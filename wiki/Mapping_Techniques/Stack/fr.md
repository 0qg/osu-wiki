# Stack

![](img/stack.jpg "Un exemple de deux hit circles empilés sur un sliderhead.")

Un **stack** est un ensemble de [hit objects](/wiki/Hit_Objects) qui sont en [overlap](/wiki/Mapping_Techniques/Overlap) les uns sur les autres dans le champ de jeu. Les objets les plus courants qui sont empilés sont les [hit circles](/wiki/Hit_Objects#hit-circle).

Les stacks sont automatiquement créées lorsque les hit objects sont proches de se chevaucher parfaitement. Le temps entre les objets pour que cela se produise est déterminé par l'indulgence du stacking de la [beatmap](/wiki/Beatmap).
