# Combo milestone

Les **combo milestones** sont un ensemble spécifique de [combos](/wiki/Glossary/Combo_(score_multiplier)) qui déclenchent le jeu pour tirer des étoiles et afficher un comboburst (si activé dans les [options](/wiki/Options)). Les combo milestones pour les modes [osu!](/wiki/Game_mode/osu!) et [osu!catch](/wiki/Game_mode/osu!catch) sont 30x, 60x, 100x, et des multiples de 50 après 100. Pour le mode [osu!taiko](/wiki/Game_mode/osu!taiko), les combo milestones sont des multiples de 50. Pour le mode [osu!mania](/wiki/Game_mode/osu!mania), les combo milestones sont des multiples de 100.
