# Le mod Random

![Icône du mod RD](/wiki/shared/mods/RD.png "Icône du mod Random (RD)")

*Pour la liste complète de tous les mods, voir : [Modificateur de jeu](/wiki/Game_modifier)*

## À propos

- Abréviation : RD
- Type : Spécial
- Multiplicateur de score : 0.00x
- Touche de raccourci par défaut : `X`
- Légende : `Mélangez les notes !`
- Modes de jeu compatibles : ![][o!m]

## Description

Le mod **Random** est un [modificateur de jeu](/wiki/Game_modifier) pour [osu!mania](/wiki/Game_mode/osu!mania) qui vise à donner aux joueurs une expérience intéressante et amusante pour osu!mania en mélangeant le placement des notes. Le mod Random ne change pas la longueur des notes, et il n'est pas mélangé à chaque fois que Random est utilisé.
