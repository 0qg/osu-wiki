# BBCode

**BBCode** — это [язык разметки](https://ru.wikipedia.org/wiki/Язык_разметки "Wikipedia"), который использует бо́льшая часть форумов в интернете. Он состоит из тегов, которые меняют стиль текста или добавляют новые элементы. BBCode применяется для оформления блоков текста в различных частях веб-сайта osu!: на форуме в постах и подписях, на юзерпейдже и в описании карт.

![Форма редактирования поста на форуме](img/editor.jpg?1 "Форма редактирования поста на форуме")

## Механизм работы

Если нажать на одну из кнопок разметки, вокруг курсора появятся открывающий и закрывающий теги разметки. Если перед нажатием на кнопку в редакторе выделить участок текста, теги появятся вокруг него.

Если необходимо применить несколько тегов к одному участку текста, можно «вложить» их один в другой. Теги при этом должны **закрываться в обратном порядке**, иначе форматирование сломается. Примеры:

- `[centre][b]текст[/b][/centre]` — правильно
- `[b][centre]текст[/b][/centre]` — неправильно

## Теги

Как и во многих других языках разметки, текст в BBCode форматируется с использованием тегов — ключевых слов, окружённых квадратными скобками (`[]`). Теги делятся на открывающие, которые активируют какой-то эффект, и закрывающие, которые его завершают. Перед названием закрывающих тегов ставится слэш (`/`).

Помимо этого, у некоторых тегов есть параметры, которые задаются после знака равенства (`=`).

Ниже приведён список тегов BBCode, работающих на сайте osu!.

### Полужирный

```
[b]текст[/b]
```

Тег `[b]` используется для изменения начертания шрифта на полужирное. Он не влияет на размер шрифта.

Кнопка: ![Кнопка полужирного шрифта](img/bold.png "Полужирный")

### Курсив

```
[i]текст[/i]
```

Тег `[i]` придаёт тексту начертание курсива, чуть наклоняя его.

Кнопка: ![Кнопка курсива](img/italic.png "Курсив")

### Подчёркивание

```
[u]текст[/u]
```

Тег `[u]` рисует под текстом горизонтальную черту, толщина которой также зависит от начертания шрифта

### Зачёркивание

```
[strike]текст[/strike]
```

Тег `[strike]` перечёркивает текст горизонтальной чертой.

Кнопка: ![Кнопка зачёркивания](img/strike.png "Зачёркивание")

### Цвет шрифта

```
[color=HEXCODE]текст[/color]
```

*См. также: [Список цветов в X11](https://ru.wikipedia.org/wiki/Список_цветов_в_X11#Список_цветов_[2] "Wikipedia")*

Тег `[color]` используется для изменения цвета выделенного текста. Выбор цвета определяется использованным [шестнадцатеричным кодом](https://ru.wikipedia.org/wiki/Цвета_HTML#Hex_triplet "Wikipedia"), либо ключевым словом вроде `red` или `green`. Чтобы поменять цвет текста, замените `HEXCODE` в примере на нужный шестнадцатеричный код. Кавычки в параметре использовать не нужно.

Если цвет не указан, то тег не будет действовать и будет оставлен как есть.

### Размер шрифта

```
[size=NUMBER]текст[/size]
```

Тег `[size]` меняет размер шрифта выделенного текста. В настоящее время сайт osu! поддерживает четыре разных размера: 50, 85, 100 и 150, которые также называются «tiny», «small», «normal» и «large». Кавычки в параметре использовать не нужно.

Если указан неподдерживаемый размер текста, или если он не указан вообще, размер текста будет стандартным.

Кнопка: ![Варианты размера шрифта](img/font-size.png "Размер шрифта")

### Спойлер

*Не путать со [спойлербоксом](#спойлербокс).*

```
[spoiler]текст[/spoiler]
```

Тег `[spoiler]` скрывает текст с помощью фона в виде чёрной полосы, чтобы его нельзя было прочитать просто так. Для прочтения скрытого текста его нужно выделить курсором. Если соединить этот тег с [`[color]`](#цвет-шрифта), то текст изменит цвет, но фон у него будет по-прежнему чёрного цвета.

Спойлеры часто используют для скрытия важной сюжетной информации о сериалах, фильмах, книгах и т.д., а также для создания комического эффекта, построенного на внезапности.

### Контейнер

*Не путать со [спойлербоксом](#спойлербокс).*

```
[box=NAME]
текст
[/box]
```

Тег `[box]` используется для скрытия текста и изображений путём помещения их в контейнер, разворачивающийся по клику.

Название контейнера задаётся аргументом `NAME` и показывается на самом контейнере. Если название не указано, контейнер можно будет открыть, только кликнув по стрелке слева. Если в названии есть несколько пробелов подряд, они будут оставлены как есть. Кавычки в параметре использовать не нужно.

Контейнеры часто используют для того, чтобы сделать пост читаемым и ограничить его размер — например, при составлении FAQ или анонсировании [скинов](/wiki/Skinning).

*Примечание: кнопка, создающая контейнер, называется «spoiler box», но не использует тег `[spoilerbox]`.*

Кнопка: ![Кнопка создания контейнера](img/spoilerbox.png "Контейнер")

### Спойлербокс

```
[spoilerbox]текст[/spoilerbox]
```

*Спойлербокс* — особый тип контейнера, где в качестве имени используется слово `SPOILER`. Для его создания используется отдельный тег, но функциональность полностью повторяет обыкновенные [контейнеры](#контейнер).

### Цитата

```
[quote="NAME"]
текст
[/quote]
```

Тег `[quote]` применяется при цитировании: слева от выделенного текста рисуется вертикальная черта и добавляется отступ. Саму цитату необходимо поместить между открывающим и закрывающим тегами, а аргумент `NAME` указывает на автора поста.

*Примечание: при замене аргумента `NAME` кавычки обязательны (`"`).*

На форумах osu! цитаты чаще всего используются для ответа на чей-то пост. Их можно добавить автоматически с помощью кнопки `Quote reply`, расположенной справа у каждого поста и видной **только** при наведении курсора на эту область.

![Кнопка ответа с цитированием](img/quotereply.png)

### Фрагмент кода

```
[code]
текст
[/code]
```

Тег `[code]` применяется для создания фрагментов текста с моноширинным шрифтом. Текст, заключённый в `[code]`, будет иметь серый фон, и никакие другие теги внутри него не будут работать.

На форумах osu! в виде фрагментов кода чаще всего оформляются [сториборды](/wiki/Storyboards) или параграфы, где необходимо подчеркнуть обязательное использование тегов.

### Центрирование

```
[centre]текст[/centre]
```

Тег `[centre]` применяется для выравнивания текста по центру. Если вместе с ним использовать `[quote]`, текст внутри цитаты будет отцентрирован, но вертикальная линия останется слева.

### Ссылка

```
[url=LINK]текст[/url]
```

Тег `[url]` используется для создания кликабельных ссылок с указанным названием.

*Примечание: `[url]` не обязателен к использованию: ссылки будут работать даже без него.*

Чтобы вставить ссылку, вместо параметра `LINK` нужно указать адрес вместе с протоколом; `текст` при этом станет названием ссылки. Если название не указано, в качестве него будет использован адрес ссылки. Кавычки в параметре использовать не нужно.

Кнопка: ![Кнопка создания ссылки](img/url.png "Ссылка")

### Профиль

```
[profile=userid]username[/profile]
```

Тег `[profile]` используется для ссылки на профиль в osu!, причём при наведении на ссылку будет показана красиво оформленная карточка игрока. Чтобы создать такую ссылку, можно использовать как ник пользователя, так и его идентификатор (параметр `userid`). При этом если ссылка задана с помощью ника, она перестанет работать или начнёт указывать на другой профиль, если игрок поменяет ник.

*Примечание: идентификатор пользователя — это число после `/users/` в ссылке на его профиль.*

Если в теге указан только идентификатор, то вместо `username` будет подставлен текущий ник игрока.

### Список

```
[list=TYPE]
[*]пункт 1
[*]пункт 2
[*]пункт 3
[/list]
```

Тег `[list]` используется для создания различных типов списков и по умолчанию создаёт ненумерованный список. Для каждого нового пункта необходимо использовать звёздочку в квадратных скобках (`[*]`).

Списки другого типа можно составить, подставив вместо `TYPE` разные типы маркеров: `1` для нумерованного списка, `a` для списка со строчными буквами, `A` — для списка с заглавными буквами, `i` или `I` — для списка с римскими числами, написанными строчными или заглавными буквами.

*Примечание: списки в BBCode можно объединять, но форматирование в них может сломаться.*

Кнопки: ![Кнопка создания списка](img/list.png "Список") ![Кнопка создания нумерованного списка](img/list-numbered.png "Нумерованный список")

### Изображение

```
[img]ADDRESS[/img]
```

Тег `[img]` используется для вставки изображений. Вместо `ADDRESS` нужно указать **прямую** ссылку на изображение из интернета (ссылки на файлы на вашем компьютере, вроде `C:\Users\Name\Pictures\image.jpg`, работать не будут).

Чтобы найти правильную ссылку, откройте страницу с изображением, кликните по нему правой клавишей и выберите `Копировать ссылку на изображение` или пункт с похожим названием.

Изображения могут находиться на любых сайтах. При этом стоит учитывать, что некоторые сайты запрещают прямые ссылки на хранящиеся на них изображения, или удаляют те с течением времени. Если требуется, чтобы картинка была всегда доступна, загрузите её на любой известный хостинг вроде [Imgur](https://imgur.com "Imgur").

Кнопка: ![Кнопка вставки изображения](img/image.png "Изображение")

### YouTube

```
[youtube]VIDEO_ID[/youtube]
```

Тег `[youtube]` используется для встраивания видео с [YouTube](https://youtube.com "YouTube") прямо в пост. Для его работы замените `VIDEO_ID` на код видео — 11 символов, которые идут сразу после `?v=` или `&v=` в ссылке на него (ссылку целиком вставлять не нужно).

### Аудио

```
[audio]URL[/audio]
```

Тег `[audio]` используется для вставки [HTML5](https://ru.wikipedia.org/wiki/HTML5 "Wikipedia")-плеера, воспроизводящего аудио. Вместо `URL` нужно указать **прямую** ссылку на аудио из интернета (ссылки на файлы на вашем компьютере, вроде `C:\Users\Name\Music\audio.mp3`, работать не будут).

*Предупреждение: далеко не все сервисы разрешают прямые ссылки на аудиофайлы, которые хранятся на них. osu! не несёт ответственности за любые проблемы, связанные с нарушением авторских прав и вызванные использованием ссылок на аудио.*

### Заголовок (вариант 1)

```
[heading]текст[/heading]
```

Тег `[heading]` используется для создания заголовков с цветным текстом. Такие заголовки имеют только один размер, и на них нельзя ссылаться.

Кнопка: ![Кнопка вставки заголовка](img/heading.png "Заголовок")

### Примечание

```
[notice]
текст
[/notice]
```

Тег `[notice]` обводит параграф с текстом и помещает его в контейнер тёмного цвета.

## Устаревшие теги

Эти теги убраны с сайта osu! и больше не работают (однако имеют некоторую историческую ценность).

### Поиск Google

```
[google]поисковый запрос[/google]
```

Тег `[google]` раньше использовался для создания ссылки на поисковую выдачу Google по определённому запросу. Как и сегодня, поисковая выдача в разных странах и у разных пользователей отличалась из-за поисковых фильтров и персонализации.

### Мне повезёт

```
[lucky]поисковый запрос[/lucky]
```

Тег `[lucky]` раньше использовался для создания ссылки на первый результат поисковой выдачи Google, имитируя нажатие на кнопку `I'm Feeling Lucky` (в русской редакции — `Мне повезёт`). Как и в случае обычного поискового запроса, адрес первого результата мог (и может) отличаться.

### Заголовок (вариант 2)

```
[текст]
```

Ранее заголовок можно было создать двумя способами: как с помощью тега `[heading]`, так и путём заключения текста в квадратные скобки. Текст, оформленный вторым образом (который, к слову, действовал только в подфоруме с картами), имел пурпурный цвет и слабое свечение.
