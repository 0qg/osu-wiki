# Программные файлы osu!

*См. также: [Форматы файлов osu!](/wiki/osu!_File_Formats)*

![Структура файлов в установочной директории osu! в Windows и macOS](img/file_structure.jpg "Структура файлов в установочной директории osu! в Windows и macOS")

**Программные файлы osu!** — это набор файлов, которые запускают osu! и отслеживают различную активность игрока во время игры.

## Пути установки

По умолчанию osu! установлена в следующих директориях:

| Windows | macOS |
| :-- | :-- |
| `C:\Users\<Username>\AppData\Local\osu!` | `/Applications/osu!.app/Contents/Resources/drive_c/osu!` |

## Папки

### Chat

Папка Chat появляется только в том случае, когда у игрока в настройках включен параметр «Вести логи личных сообщений» или если игрок выполнит команду `/savelog` в [окне чата](/wiki/Chat_Console).

Структура имени файлов: `{Название_вкладки}-{ГГГГММДД}-{ЧЧММСС}`, их можно открыть в любом текстовом редакторе. Ниже показан пример этого:

``#multiplayer-20121115-040845 (/savelog во вкладке #multiplayer 15 ноября 2012 в 04 часа 08 минут и 45 секунд).``

### Downloads

В папке Downloads хранятся карты, скачиваемые через osu!direct (для этого нужен [osu!supporter](/wiki/osu!supporter)). После скачивания они автоматически переносятся в папку Songs.

### Exports

Папка Exports появляется, когда игрок использует опцию [«Экспортировать в .osk» в меню выбора скинов](/wiki/Options) или [«Экспортировать в .osz» в редакторе карт](/wiki/Beatmap_Editor/Menu). В ней будут карты и скины, которые игрок экспортировал из osu!.

### Localisation

Папка Localisation появляется, когда игрок переключил язык в настройках игры. В ней будут переведённые текстовые файлы, каждый из которых может использоваться для замены английского текста на основе выбранной пользователем локализации. Переведённые текстовые файлы создаются при переключении языков.

### Replays

*Примечание: раньше реплеи записывались с низкой частотой кадров, но потом её увеличили для более плавного воспроизведения.*

В папке Replays хранятся файлы реплеев игрока. Файл реплея не работает, когда связанные с ним карты отсутствуют. Реплей содержит результаты игрока, а также оживляет движения курсора игрока во время воспроизведения. Чтобы создать реплей, нажмите F2 на экране с результатами или кликните по кнопке «Сохранить реплей» (только в одиночном режиме).

*Игрокам, которые хотят загрузить свои реплеи на YouTube, см.: [публичный релиз Osr2mp4. Автоматическая конвертация файла с реплеем в видео.](https://osu.ppy.sh/community/forums/topics/1104243)*

Структура имени файла: `{Никнейм локального игрока} - {Исполнитель} - {Название} {[Сложность]}{(ГГГГ-ММ-ДД)} {Игровой режим}`. Ниже показан пример этого:

``dummytest1 - Loituma - Ievan Polkka \[SPINNER-MADNESS\]  (2013-08-12) OsuMania``

### Screenshots

В папке Screenshots хранятся скриншоты, созданные игроком в osu!. По умолчанию скриншот сохраняется в расширении файла `.jpg`, однако его можно изменить на `.png` в настройках.

*Заметка: чтобы сделать скриншот, нажмите клавишу скриншота (по умолчанию F12).*

Структура имени файла: `screenshot###`, где "###" — номер скриншота.

### Skins

В папке Skins хранятся скины, созданные игроками, которые можно использовать для изменения игрового интерфейса. Игроки могут скачать скины с [подфорума скинов](https://osu.ppy.sh/community/forums/15). Их можно установить, кликнув два раза по скину в файловом менеджере. «osu! by peppy» — единственный скин, который не имеет папки и нельзя удалить.

*См. также: [Создание скинов](/wiki/Skinning)*

### Songs

*Предупреждение: будьте осторожны с этими файлами. Вы можете сломать osu!, если не будете осторожны.*

В папке Songs хранятся пользовательские карты osu!. Папка с картой обычно содержит: `.osu` (сложности), `.mp3`/`.ogg` (аудиофайлы), `.jpg`/`.png`/`.gif` (фоновые изображения), `.osb` (файлы со сторибордом) и `.mp4`/`.flv` (видеофайлы). Также может содержать `.wav`/`.ogg` (файлы с хитсаундами) и папки (с раскадровкой сториборда и/или папкой со скином).

Структура имени файла: `{Номер карты} {Исполнитель} - {Название песни}`.
**Пример:** [57950 SOUND HOLIC - Drive My Life](https://osu.ppy.sh/beatmapsets/57950)

Обратите внимание, что очень старые карты (например, [Kenji Ninuma - DISCO PRINCE](https://osu.ppy.sh/beatmapsets/1) или [Dudelstudios - Angry Video Game Nerd Theme [MATURE CONTENT]](https://osu.ppy.sh/beatmapsets/66)), так же как и незагруженные карты, не соответствуют этому формату.

## Скрытые папки

Эти папки скрыты, поскольку любые изменения в них могут помешать правильному запуску osu!.

### Data

Файлы данных osu!. Содержат некоторый кэш osu!, такие как кэши фоновых изображений карты и аватаров.

## Файлы

*Предупреждение: будьте осторожны с этими файлами. Вы можете сломать osu!, если не будете осторожны.*

## Файлы базы данных (.db)

Файлы базы данных — базы данных, которые требуются osu!, чтобы правильно работать. В файлах содержатся важная информация, которая требуется osu!, например: сохранённые скоры и кэшированный список карт, сохранённых на устройстве игрока.

- `collections.db`: хранит внутриигровые «коллекции» игрока.
- `osu!.db`: хранит базу данных карт osu!.
- `presence.db`: хранит кэш игроков osu!, зашедших в окно чата.
- `scores.db`: хранит локальные таблицы рекордов.

## .cfg (Файлы конфигурации)

Файлы конфигурации настраивают начальные параметры для работы osu!. Эти файлы можно открыть с помощью текстового редактора.

- `osu!.cfg`: хранит информацию о безопасности файлов osu! и текущую ветку обновлений. Этот файл никогда не должен изменяться вручную.
- `osu!.<имя пользователя в операционной системе>.cfg`: хранит данные [настроек](/wiki/Options) и другие игровые настройки. См. [Файл конфигурации](/wiki/osu!_Program_Files/User_Configuration_File).

### .exe (Приложение)

Главный компонент. Кликните по нему для запуска (работает только на Windows). Файлы .exe можно безопасно открывать, если игрок для установки osu! использовал установщик osu!, скачанный с официального сайта.

osu!.exe (Запускает osu!)

#### .dll (Расширения приложения)

Файлы .dll — зависимости и компоненты osu!.
