Syntax
------

    {{BigBox
    |border = 
    |background = 
    |color = 
    |min-width = 
    |image = 
    |pic-align = 
    |caption = 
    |text = 
    |cat1 = 
    |cat2 = 
    |cat3 = 
    }}

-   border — If a value is given, the given **style** will be used for the border
-   background — If a value is given, the given **style** will be used for the background
-   color — If a value is given, the given **style** will be used for the text color
-   min-width — If a value is given, the given **style** will be used for the minimum width
-   image — If a value is given, the given **file name** will be used for the image
-   pic-align — If a value is given, the given **alignment** will be used for the image
-   caption — If a value is given, the given **caption** will be used for the image
-   text — If a value is given, the given **text** will be used for the text
-   cat1 — If a value is given, the given **category** will be used as the category
-   cat2 — If a value is given, the given **category** will be used as the category
-   cat3 — If a value is given, the given **category** will be used as the category

Usages
------

### Normal

    {{BigBox|text=XYZ}}

-   Replace **XYZ** with your text
