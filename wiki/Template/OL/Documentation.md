***Update**: This template was fixed!* Use **{{OL|**XX**}}** on any article (*XX* = LANGUAGE CODE)

This template will link to articles in this wiki that are in another language.

To use the template, add `{{OL}}` to the top of the article and you are ready to go. However, using the template in this way can (and will) limit how the article in another languages are named (The article can't be named differently except for the language prefix), so for this, you can specify a different article name for each language and still be linked in the template.

To use this, when adding the template, add the language code and the article name to it, like this hypothetical example of [FAQ](FAQ "wikilink") links:

    <nowiki>{{</nowiki>OL
    |es=Preguntas Frecuentes
    |ru=FAQ
    |fi=Insert translation of FAQ here
    <nowiki>}}</nowiki>

If you add this in [FAQ](FAQ "wikilink"), the *Other Languages* section will link to [ES:Preguntas Frecuentes](ES:Preguntas_Frecuentes "wikilink") for the Spanish link, <RU:FAQ> for the Russian link and [FI:Insert translation of FAQ here](FI:Insert_translation_of_FAQ_here "wikilink") for the Finnish link.
