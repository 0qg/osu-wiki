This is a basic template for navigation ones.

Parameters
----------

-   **name** — required, must be equal of the name of the template
-   **image**
-   **imagewidth**
-   **imagelink**
-   **bgcolor**
-   **titlebgcolor**
-   **titlecolor**
-   **ghbgcolor**
-   **ghcolor**
-   **goddstyle**
-   **gevenstyle**

Example
-------

    {{Group
    | name         = Group/Documentation
    | title        = My school

    | image        = logo.png
    | imagesize    = 32px

    | bgcolor      = lightblue
    | titlebgcolor = #000000
    | titlecolor   = yellow
    | ghbgcolor    = #79bacf
    | ghcolor      = red

    | group01      = Classes
    | text01       = [[9A]] • [[9В]] • [[10A]] • [[10B]] • [[10C]] • There's more, but they are 'tards

    | group02      = Teachers
    | text02       = [[Mary Sergevna]] • [[John Smith]] • [[Alex “Bull” Aleksandrovitch]]

    Till 20
    }}
