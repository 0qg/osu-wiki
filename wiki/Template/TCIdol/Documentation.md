**T**able extension for **C**ontests' osu!**idol**.

`{{TCIdol|tF=Full|tNP=News` `post|tRP=Results` `post|tOT=Official` `thread` `|1|img=` `|a1=|s1=|oa1=` `|a2=|s2=|oa2=` `|a3=|s3=|oa3=` `|FL=` `|NP=|RP=|OT=}}`

Translation (t):-

-   tF = Full
-   tNP = News post
-   tRP = Results post
-   tOT = Official thread

1. Year
img = Image for osu!idol. Please append the image extension (idol2014.png)
a = artist (use `{{Profile}}`), s = song, oa = original artist.
FL = Full list link desc.
NP = Link number. (1234)
RP = Link number. (1234)
OT = Link number. (t/1234 or p/1234) <noinclude></noinclude>
