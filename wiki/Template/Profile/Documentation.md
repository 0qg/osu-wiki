This template is for generating a link to the user's profile without showing the default external link icon beside it.

When to use
-----------

When adding user profiles on articles.

How to use
----------

To use this template, type

    {{Profile|<UserID>|<Name>|<color>|<flag>}}

on the article.

-   Remember to edit "<UserID>" to the actual user's ID and "<Name>" to any name.

When just typing `{{Profile}}`, it will appear as

### Colours

Example
-------

Code: `{{Profile|2295078|Nathanael|green|ph}}`
