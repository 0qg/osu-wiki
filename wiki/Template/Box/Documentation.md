This template is a base for the rest of the templates used in the wiki that require it.

When to use
-----------

This template is usually a base, so normal usage is not recommended.

How to use
----------

The simplest way to add this template to an article is to copy and paste either `{{Box|text=Your` `text` `here.}}` at the top of the article and edit the `Your` `text` `here.` part to the text of your liking.

Image Parameter
---------------

You may use an image parameter with this template, for example `{{Box|image=foo.png}}` will change the image in the box to the image foo.png in the wiki. If you leave it out, the image Alert.gif will be used. You can set the width of the image with `{{Box|iwidth=25}}` (This example sets the width of the image at 25px), alignment of the image with `{{Box|pic-align=(center|left|right)}}` and caption with `{{Box|caption=This` `is` `my` `caption` `for` `the` `box` `image!}}`.

Color Parameter
---------------

You may use a color parameter with this template, for example `{{Box|color=white}}` to change the color of the box to white. If you leave it out, a gray color will fill it in. Be aware that any color supported in CSS can be used in the `color` parameter.

Border Color Parameter
----------------------

You may use a border color in this template, for example `{{Box|border=red}}` to change the color of the box border to red. If you leave it out, the border will be transparent. Be aware that any color supported in CSS can be used in the `border` parameter.

Width Parameter
---------------

You may set a set width for the box using the width parameter, for example `{{Box|width=350px}}` will set the box width to 350 pixels. If you leave it out, the box will no lock itself at a set width. Note: You can use values like `{{Box|width=13em}}` or `{{Box|width=80%}}`.

Categorization
--------------

This template supports categorization from its parameters. You can add up to 3 categories when using this template. The parameters are `cat1`, `cat2` and `cat3`. The templates are added to the page using them.
