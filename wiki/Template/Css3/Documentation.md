Template for adding vendor prefixes (*-moz-*, *-ms-*, *-o-*, *-webkit-*, and the standard).

Syntax
------

    {{Css3|1}}

-   *parameter 1* = CSS styling code

Example
-------

    <div style="{{Css3|background:red;}}">I'm a &lt;div&gt; with a red background!</div>

Yields:

<div style="{{Css3|background:red;}}">
I'm a &lt;div&gt; with a red background!

</div>
