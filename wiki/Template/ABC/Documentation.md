This is a template for contents. \_\_NOTOC\_\_

Syntax
------

    {{ABC
    |noabc = 1
    |noabv = 1
    }}

-   `noabc` — If set to *1* all latin characters will be removed
-   `noabv` — If set to *1* all cyrillic characters will be removed

Pseudo templates
----------------

*Copy and Paste ready!*

### Normal

    {{ABC}}

### No latin characters

    {{ABC|noabc=1}}

### No cyrillic characters

    {{ABC|noabv=1}}
