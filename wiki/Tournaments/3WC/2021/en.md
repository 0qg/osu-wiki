# 3 Digit World Cup 2021

![3WC 2021 Banner](img/banner.png)

The **3 Digit World Cup 2021** (***3WC 2021***) is a country-based osu! tournament hosted by ![][flag_NL] [nik](https://osu.ppy.sh/users/10077264) and ![][flag_FI] [shdewz](https://osu.ppy.sh/users/10000899). The tournament is for players ranked between #100 and #999. It is the 3rd installment of the 3 Digit World Cup.

## Tournament schedule

| Event | Timestamp |
| --: | :-- |
| Registration phase | 2020-02-24/2020-03-13 (18:00 UTC+0) |
| Team submission | 2020-03-13/2020-03-28 (18:00 UTC+0) |
| Qualifier stage | 2020-04-02/2020-04-04 |
| Round of 16 | 2020-04-09/2020-04-11 |
| Quarterfinals | 2020-04-16/2020-04-18 |
| Semifinals | 2020-04-23/2020-04-25 |
| Finals | 2020-04-30/2020-05-02 |
| Grand Finals | 2020-05-07/2020-05-09 |
