# Редактор карт

У osu! есть встроенный **редактор** для создания собственных карт. Он состоит из четырёх основных разделов, каждый из которых представлен отдельной вкладкой:

1. [Compose](Compose): здесь создаются **паттерны** из кружков и слайдеров. Большую часть времени мапперы проводят именно тут.
2. [Design](Design): раздел для добавления фоновой картинки или видео, а также редактирования **[сториборд](/wiki/Storyboards)** — разнообразной анимации, которая прокручивается на фоне во время игры.
3. [Timing](Timing): выставление оффсета и BPM песни, чтобы ноты попадали в ритм. Ещё здесь можно настроить **[хитсаунды](/wiki/Guides/Custom_Sample_Overrides)** — звуки попаданий по игровым объектам.
4. [Song Setup](Song_Setup): настройка игровых параметров карты (например, размер нот), а также названия песни и прочих **метаданных.**

Кроме перечисленного, в редакторе есть ещё несколько компонентов и понятий:

- [Меню](Menu), скрывающее в себе множество инструментов и настроек;
- [AiMod](AiMod) — набор автоматических проверок для карты;
- [Временная шкала](Timelines), где расположены игровые объекты;
- [Деления временной шкалы](Beat_Snap_Divisor), влияющие на верхнюю линейку с нотами;
- [Спейсинг](Distance_Snap), отвечающий за расстояние между нотами;
- [Киаи](Kiai_Time), или вспышки и фонтаны из звёзд для «мощных» участков песни;
- [Нагрузка от сториборда](SB_Load), показывающая, как сильно фоновая анимация влияет на производительность игры;
