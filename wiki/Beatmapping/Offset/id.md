# Offset

Sebuah [beatmap](/wiki/Beatmaps) **offset** adalah jumlah dari waktu antara dimulainya file audio dan downbeat pertama dari musik yang diukur dalam miliddetik. Offset digunakan untuk menentukan [timing](/wiki/Beatmap_Editor/Timing) dari sebuah beatmap,  ais the amount of time between the start of the audio file and the song's first downbeat measured in milliseconds. Offset is used to determine a beatmap's [timing](/wiki/Beatmap_Editor/Timing), disamping [beats per minute (BPM)](/wiki/Beatmapping/Beats_per_minute) dan [meter](https://en.wikipedia.org/wiki/Metre_(music)).

Video termasuk dengan [beatmapsets](/wiki/Beatmaps/Beatmapsets) juga memiliki offset yang dapat di set sedemikian rupa dengan offset musik asli.
