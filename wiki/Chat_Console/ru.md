# Окно чата

В osu! есть внутриигровой чат, которой можно открыть, нажав `F8` или кликнув по кнопке `Show Chat` в правом нижнем углу экрана.

![Окно чата](img/Chatconsole1.png "Окно чата")

- Открытые каналы чата представлены вкладками. Для переключения на другой канал кликните по вкладке с соответствующим названием. Чтобы увидеть список всех доступных каналов, кликните по жёлтому плюсу.
- В зависимости от ролей, ники пользователей имеют разные цвета.

| Цвет | Роль |
| :-- | :-- |
| **Белый** | Вы |
| **Бледно-жёлтый** | Игроки, не поддержавшие osu! |
| **Жёлтый** | [Саппортер](/wiki/osu!supporter) |
| **Красный** | [Модератор](/wiki/People/The_Team/Global_Moderation_Team) или член [команды оценки номинаций](/wiki/People/The_Team/Nomination_Assessment_Team) |
| **Зелёный** | Сообщение содержит Ваше имя пользователя или слово которое Вы добавили в список подсвечиваемых в чате слов. Копия сообщения так же появится в канале `#highlight`. |
| **Синий** | Личное сообщение |
| **Голубой** | [peppy](https://osu.ppy.sh/users/2), создатель osu! |
| **Розовый** | [BanchoBot](/wiki/BanchoBot) |

- Кликните на `Show Ticker` для отображения новых сообщений в нижней части экрана, когда окно чата скрыто.
- Кликните на `Auto-Hide`, чтобы автоматически скрывать окно чата во время игры (не работает во время вступления к карте, её концовки и перерывов).
- Кликните на `Hide Chat` или нажмите `F8`, чтобы скрыть окно чата.

## Расширенное окно чата

На большинстве экранов в osu! можно нажать `F9` или кликнуть `Online Users` в нижнем правом углу для открытия расширенного окна чата, в дополнение к окну чата, расширенное окно чата показывает на оставшейся части экрана список пользователей, которые зашли в osu!.

![Расширенное окно чата](img/Chat_Console-Extended.png "Расшриенное окно чата")

В расширенном окне отображаются панели всех игроков, находящихся в сети. По умолчанию панель отображает основные сведения об игроке (имя, общий рейтинг, ранк, точность, количество игр и аватар, если он есть). Если навести на неё курсор, будут видны дополнительные данные (имя, ранк, аватар, если есть, время и часовой пояс, местоположение и страна, город (при согласии пользователя) и текущая активность).

- `Все` - отображает всех. `Друзья` - отображает только Ваших друзей. `Рядом` - отображает игроков в ближайших городах от вашего. `Страна` - отображает игроков из Вашей страны.
- `Зафиксировать панели` предотвращает перемещение панелей при входе/выходе пользователей.
- Нажмите на вкладки сверху чтобы отсортировать пользователей по определённым критериям.
- Нажмите на `Карта мира` чтобы посмотреть карту мира с отображением где находятся пользователи, которые сейчас в сети.
- Вы можете нажать и перемещать белый прямоугольник для прокрутки, так же Вы можете это сделать с помощью колёсика мыши.
- Пользователи без статистики на панели вошли в чат с помощью IRC клиента.

| Цвет панели | Описание |
| :-- | :-- |
| ![Игрок ничего не делает](img/Chat_Console-Idle.png "Игрок ничего не делает") | Тёмно-синий - Игрок в данный момент ничего не делает или просто общается в чате. |

| ![Игрок тестирует карту](img/Chat_Console-Testing.png "Игрок тестирует карту") | Фиолетовый - Игрок тестирует (в игре) чужую карту. |

Нажатие на панель пользователя открывает экран с опциями

![Экран с опциями](img/Chat_Console-Userpaneloptions.jpg "Экран с опциями")

1. `Наблюдать за игрой`: Если пользователь играет карту, Вы можете посмотреть за его игрой. Ваше имя пользователя будет отображено в списке наблюдателей у этого игрока.
2. `Открыть профиль`: Открывает профиль этого игрока в браузере.
3. `Начать разговор`: Открывает вкладку чата с данным пользователем.
4. `Пригласить в мульти`: (Если Вы в мультиплеерной комнате) Приглашает пользователя присоединиться к комнате.
5. `Добавить (Удалить) из друзей`: Добавляет (Удаляет) пользователя из Вашего списка друзей.
6. `Отправить жалобу...`: Сообщает о плохом поведении пользователя. Не должна использоваться для чего-то другого. Вы можете пожаловаться на пользователя в игре, но так же можете сделать сделать это на сайте, например на форуме.
7. `Игнорировать`: Скрывает сообщения данного пользователя в Вашем окне чата.
8. `Отмена`: Закрывает данный экран.

## Список команд

### /help

| Команда | Результат | Пример | Ответ BanchoBot'а |
| :-- | :-- | :-- | :-- |
| `/addfriend [пользователь]` | Добавляет `[пользователь]` в Ваш список друзей. | `/addfriend Amigo` | You are now friends with Amigo. |
| `/delfriend [пользователь]` | Удаляет `[пользователь]` из Вашего списка друзей. | `/delfriend Amigo` | You are no longer friends with Amigo. |
| `/away [сообщение]` | Изменяет сообщение, отправляемое когда Вы отошли от клавиатуры (отправляется когда пользователи отправляют Вам личное сообщение). Оставьте `[сообщение]` пустым чтобы удалить его. | `/away I am John Smith.` | You have been marked as being away: I am John Smith. Когда другой пользователь отправит вам сообщение BanchoBot ответит пользователю I am John Smith. |
| `/bb` | Отправляет сообщение BanchoBot'у чтобы выполнить команду, например `!stats [пользователь]` | `/bb !stats Uan` | \[15/11/12\] Stats for [Uan](https://osu.ppy.sh/users/147623): Score: 47,323,299,680 (#1) Plays: 176293 (lv102) Accuracy: 98.95% |
| `/chat [пользователь]` | Открывает вкладку чата с данным пользователем. | `/chat Amigo` | (Открылась вкладка чата с Amigo) |
| `/clear` | Очищает историю чата. | `/clear` | (Очищает всё в данной вкладке) |
| `/ignore [пользователь][@chp]` | Скрывает все сообщения от указанного пользователя. Добавляя @ с буквами, c, h, и/или p, Вы можете скрыть его сообщения в чате, упоминаниях или личных сообщениях соответственно. | `/ignore Amigo@chp` | You will no longer hear Amigo {chat} {highlights} {PM} |
| `/j [канал]` или `/join [канал]` | Присоединение к указанному каналу | `/join #lobby` | (Открывается вкладка #lobby) |
| `/p` или `/part` или `/close` | Выход из текущего канала. | `/part` | (Вкладка с текущим каналом закрылась) |
| `/unignore [пользователь]` | Снова показывает сообщения от указанного пользователя. | `/unignore Amigo` | Теперь до Вас будут доходить сообщения Amigo. |
| `/me [действие]` | Выполняет действие от третьего лица. | `/me is at home` | * John is at home |
| `/msg [пользователь] [сообщение]` | Отправляет личное сообщение `[пользователь]`. | `/msg Amigo I am sick at home.` | (Во вкладке с Amigo) John: I am sick at home. |
| `/np` | Отправляет в чате карту, в которую Вы играете, слушаете или смотрите повтор. | `/np` | (Если играете) * John is playing [Peter Lambert - osu! tutorial \[Gameplay Basics\]](https://osu.ppy.sh/beatmapsets/3756#osu/22538) |
| `/reply` или `/r` | Отвечает на последнее сообщение. | `/r Do you know any good doctor?` | (Во вкладке с Amigo) John: Do you know any good doctor? |
| `/savelog` | Сохраняет историю чата в текстовый файл. | `/savelog` | (Создаётся текстовый файл в папке Chat с историей чата) |
| `/watch [пользователь]` | Начинает наблюдение за игрой `[пользователь]`. | `/watch Amigo` | * Started spectating Amigo. |
| `/nopm` | Разрешает/запрещает отправлять Вам личные сообщения пользователям не из Вашего списка друзей | `/nopm` | (Если доступны для всех) Личные сообщения не от друзей будут игнорироваться. |
| `/invite [пользователь]` | Приглашает `[пользователь]` в мультиплеерную комнату по ссылке. | `/invite Nathanael` | Вы приглассили Nathanael в вашу комнату. |

## /keys

| Кнопки на клавиатуре | Результат |
| :-- | :-- |
| `Page Up` / `Page Down` | Прокручивает окно чата. Для этого так же можно использовать колёсико мыши. |
| `Tab` | Автозаполнение вводимого имени пользователя. |
| `F8` | Показать/скрыть окно чата. |
| `F9` | Показать/скрыть расширенное окно чата. |
| `Ctrl` + `C` / `V` | Копирование/Вставка. |
| `Alt` + `0` - `9` | Перейти к соответствующей вкладке. |
